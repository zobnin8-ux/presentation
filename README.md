# GRC US Presentation Site

Animated one-page presentation for GRC US market automation system.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

```bash
npm run build
# Vercel: vercel --prod
```

## Customize

Edit contact info in `lib/content.ts` → `siteConfig.author`
